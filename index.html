Workflow Overview

This would be the landing page for a coordinator, engineer, or quality lead. The idea is one job number drives everything else.

1. Load Job

Search or enter a job number. Pull customer, part, revision, and due date from your source system.

2. Build / Update F738

Open the controlled F738 record, edit approved fields, and export a clean output for release.

3. Complete Checklist

Track the actual quality/process checklist against the same job so nothing is split across systems.

4. Pull Standards + Quick Reference

Search standards by customer/category and generate a condensed quick-reference sheet for the team.
